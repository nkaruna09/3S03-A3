### Running Test Cases for Question 4a

Before running the tests, ensure that the Selenium server is running in the background.

1. Navigate to the project directory:

   ```
   cd SelenumTests_Q4a
   ```

2. Clean the project:

   ```
   mvn clean
   ```

3. Run the test suite:

   ```
   mvn test
   ```
